Lagan Template Installer Plugin for Composer
============================================

Composer plugin to install Lagan admin content templates in their own property-templates directory outside the default vendor directory.

To be used with [Lagan](https://github.com/lutsen/lagan). Lagan lets you create flexible content objects with a simple class, and manage them with a web interface.
